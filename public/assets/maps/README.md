# Tamil Nadu Election Map

## Required File

This folder should contain:
- `tn-election-map-2021.png` - Official 2021 Tamil Nadu Legislative Assembly Election Map

## Download Instructions

### Option 1: Wikimedia Commons (Recommended)
1. Visit: https://commons.wikimedia.org/wiki/File:2021_Tamil_Nadu_Legislative_Assembly_election_result.png
2. Click the download button or right-click the image and "Save image as..."
3. Save the file as `tn-election-map-2021.png` in THIS directory
4. Refresh the Tamil Nadu Map Dashboard page

### Option 2: Election Commission of India (Official)
1. Visit: https://eci.gov.in/files/file/6852-election-map-tamil-nadu/
2. Download the official election map
3. Save it as `tn-election-map-2021.png` in THIS directory
4. Refresh the Tamil Nadu Map Dashboard page

## File Location

Place the downloaded file here:
```
frontend/public/assets/maps/tn-election-map-2021.png
```

## Map Specifications

- **Format**: PNG (recommended for best quality)
- **Name**: Must be exactly `tn-election-map-2021.png`
- **Content**: 2021 election results showing all 234 Tamil Nadu assembly constituencies
- **Dimensions**: At least 1200px width recommended for clarity
- **License**: CC BY-SA 4.0 (from Wikimedia Commons)
